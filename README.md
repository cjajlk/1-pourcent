# The 1 Percent Club

> Projet web dédié au concept du "1 Percent Club".

---

## Présentation

Ce projet propose une page HTML mettant en avant le concept du "1 Percent Club". Il s’agit d’un espace digital pouvant servir de vitrine, de présentation ou de point de départ pour des initiatives liées à l’amélioration continue et à l’excellence.

## Fonctionnalités
- Page web statique
- Design personnalisable
- Structure simple et adaptable

## Structure du projet
- `the-1-percent-club.html` : page principale du site

## Installation & Utilisation
1. Téléchargez ou clonez le projet.
2. Ouvrez le fichier `the-1-percent-club.html` dans votre navigateur web préféré.
